=== Old MacOS for Windows 10/11 Cursor Set ===

By: blueslime32

Download: http://www.rw-designer.com/cursor-set/old-macos-win10-11

Author's description:

These old MacOS cursors were remade by me from the ground up to suit Windows 10/11's high display resolutions! They look very nice if you're looking for an old skeuomorphic/aero aesthetic for your computer!

To use the higher scale versions of these cursors, you might want to first go to Settings > Ease of Access > Mouse pointer and use the slider under "Change pointer size" up to 2 bars to use the different sizes! (Windows 11 instructions might be similar, I don't use Windows 11).

For some reason, RealWorld didn't like the Busy cursor, so I will be hosting a download for it in this link: https://www.mediafire.com/file/uq2mvclgm82my5b/Busy.ani/file

Thank you for using! :)
